Description

Drawing Obstacles

You can draw obstacles on the grid by pressing and holding the mouse button. These obstacles will block the algorithm's pathfinding.

Start

The "Start" button initiates the algorithm, allowing it to begin processing and finding the solution.

Stop

The "Stop" button pauses the algorithm's execution. If the algorithm is already stopped, pressing "Stop" will resume its execution.

Clear

The "Clear" button removes any obstacles that have been drawn on the grid, allowing you to reset the environment.

Restart

The "Restart" button resets the entire grid, including obstacles and the algorithm's state, bringing it back to the initial state.
